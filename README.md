# DateCaculate
**说明**：本项目是一个结构清晰、功能聚焦的 Java 日期计算示例程序，非常适合**初学者**用于学习和理解以下核心知识点：
*   **Java 基础语法**与**面向对象**编程思想。
*   如何组织一个包含多个类的小型项目结构。
*   **`java.util.Date`、`Calendar` 或 `java.time` (如果使用 Java 8+)** 等日期时间 API 的实际应用。
*   计算两个日期之间的差值（天数、月数等）的核心逻辑。
*   基本的**命令行程序**的编写与交互。
通过阅读和运行本项目的代码，你可以直观地看到一个完整的小工具是如何从设计到实现的。

## 项目描述
本项目包含两个核心的日期计算应用，旨在提供便捷的日期差计算、日期格式化以及相关时间操作功能。

## 主要功能
*   **日期距离计算**：计算两个给定日期之间的天数、月数或年数差。
*   **日期格式化**：提供统一的日期格式协议，确保日期在不同模块间的一致性。
*   **定时/计时计算**：可能涉及与时间点或时间段相关的计算功能。

## 项目结构
```
DateCaculate/master
├── DateCaculateApp.java      # 主日期计算应用逻辑
├── DateFormatProtocol.java    # 日期格式定义与协议
├── MainApp.java              # 应用程序主入口
├── PublicActionProtocol.java # 公共操作协议或接口
└── TimerCaculateApp.java     # 定时/计时计算应用逻辑
```

## 使用方法
1.  **环境准备**：确保你的系统已安装 Java 运行环境 (JRE) 或 Java 开发工具包 (JDK)。
2.  **编译项目**：在项目根目录下，使用以下命令编译所有 `.java` 文件：
    ```bash
    javac *.java
    ```
3.  **运行程序**：通过以下命令启动应用程序。
    ```bash
    MainApp.java
    ```
